Academic Utility App

This web application helps students calculate CGPA, compute final grades, and predict outcomes based on partial assessment data. It is built using simple HTML, CSS, and JavaScript, with all calculations performed directly in the browser.

The tools are structured around a multi-level academic program and are designed to handle real grading rules, credit-weighted CGPA, and incomplete courses accurately.
